# Exoplanet Detection using Machine Learning

## Project Overview

This project uses machine learning techniques to detect **exoplanets** by analyzing changes in the brightness of stars over time.

When a planet passes in front of a star, the observed brightness slightly decreases. This phenomenon is known as the **transit method**. By analyzing these small dips in brightness, it is possible to detect potential planets orbiting distant stars.

## Dataset

The dataset used in this project is based on astronomical observations from the Kepler Space Telescope and is available on Kaggle.

The dataset contains measurements of star brightness over time, along with labels indicating whether a potential exoplanet is present.

## Objectives

The goals of this project are:

* Analyze stellar brightness data
* Detect patterns indicating planetary transit
* Train a machine learning classifier
* Predict whether a star hosts an exoplanet

## Machine Learning Models

The following models are explored:

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)

The model successfully learns to distinguish between stars with and without exoplanet transit signals by analyzing small variations in brightness.
